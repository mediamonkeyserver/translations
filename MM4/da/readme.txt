MediaMonkey Readme 
November 8, 2010 
------------------------- 

1. Indledning 

MediaMonkey er en nem at bruge medierne arrangør for seriøs musik samlere. Det kataloger
musikfiler (MP3, WMAs, OGG, AAC/M4A, MPC, APE, FLAC, VQF, WAV-filer) og lyd-cd'er, og hjælper dig med:
 
  - Organiser din musik, og hjælpe dig med at finde de numre, du leder efter 
  - Tag spor i biblioteket (automatisk) i en række formater (ID3v1 & v2, OGG, WMA, APE2, RIFF)
  - Afspil MP3 og andre lydformater, niveau volumen, og vise funky visualiseringer
  - Synkroniser spor med din bærbare lydafspiller 
  - Optag cd'er til OGG, MP3 og WMA filer 
  - Konverter enhver understøttet format til ind OGG, WAV, MP3 eller WAV-filer 
  - Oprette spillelister og musik blander hurtigt (manuelt eller automatisk) passer til enhver lejlighed
  + Meget, meget mere ... 


2. OS Krav 

MediaMonkey er understøttet på: 
  - Windows 7 
  - Windows Vista 
  - Windows XP 

MediaMonkey er også kendt for at arbejde, men er understøttet på: 
  - Windows 2000 (SP3/SP4) 
  - Windows 98 SE 
  - Windows ME 
  - VIN (Linux) 


3. Installation 

a) Dobbeltklik på den fil, du downloadede, og følg instruktionerne på skærmen. 

b)  Hvis du havde en tidligere version af MediaMonkey Guld installeret, og slettede det før
    installere den nye version af MediaMonkey, så konfigurationsindstillinger din Virtual CD
    vil gå tabt. Gå til: Funktioner | Indstillinger | Virtual CD & Preview, og nulstille den virtuelle cd bibliotek
    til sine tidligere placering. 

c)  Hvis ingen lyd høres, når du forsøger at afspille et musiknummer, kan du prøve at skifte output plug-ins.
    Gå til Funktioner | Indstillinger | Spiller | Output Plug-ins og skifte fra DirectSound output til
    WaveOut output. 

d)  Hvis du har problemer med at spille eller kodning WMA (Windows Media Audio) filer, kan det
    skyldes forældet Windows Media Drivers. Du kan installere de nyeste drivere fra:
    http://www.microsoft.com/windows/windowsmedia/format/codecdownload.aspx 

    Bemærk: MediaMonkey understøtter WMA DRM 9, men understøtter endnu ikke filer beskyttet af WMA DRM 10.
 

4. Advanced Configuration 

a)  Kommandolinjevalg 
    En række af flag kan indstilles, når du kører MediaMonkey fra kommandolinjen: 
    /NoSplash                 - ikke vise velkomstbilledet ved opstart MediaMonkey 
    /Næste                    - spil filer efter det aktuelle spor 
    /Tilføj                   - tilføje filer til Spiller nu køen 
    /Party                    - opstart i Party Mode 
    /Spil                     - opstart og spille, hvad er i øjeblikket i Spiller nu køen 
    /Nulstil værktøjslinje    - nulstiller værktøjslinje konfiguration til standardindstillingerne
 
    Du kan bruge disse muligheder som følger: 
    MediaMonkey /NoSplash /Tilføj "Min Sang.mp3"
